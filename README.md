# 🎮 Gun-Fight-2000
Play at: https://jonathanelazar.itch.io/gun-fight-2000

A locally run 2 player game where you have to beat the other player in a gun fight.

## ⬇️ Download Guide
This can be downloaded by compiling the source code with godot engine or you can use itch.io to download or play online.
### 📂 Build with Godot
  1. Download the source code of the main branch.
  2. Open Godot Engine and press import project then navigate to the downloaded source code.
  3. Than navigate to Project -> Export
  4. Select the platform you are using and press export.
### 📁 Download with itch.io (or play online)
  **I dont have a itch.io build for macOS if you are using this operating system you must play online or build it yourself with godot**
  1. Go to https://jonathanelazar.itch.io/gun-fight-2000
  2. Click on play game to play online or scroll down to windows.zip and linux.zip and download the one that matches your operating system.

## 🎲 How to play
When you open the game you will see two characters one for each player. The player using the character to the left uses WASD and the player using the character to the right will use arrow keys.

<img width="320" height="186" alt="mygame" src="https://github.com/user-attachments/assets/4b4d9810-7e19-428b-8d90-a15e0e242f78" />

Your up key will let you jump your left and right keys let you move left and right and your down key shoots a bullet.
You win the game if you bullet hits the other player.

## 🍎 macOS users
If you are using macOS and build the game yourself please upload the build as an issue so other macOS users can use your build.
